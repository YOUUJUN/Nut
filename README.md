## 项目指令：
项目初始化：
`cd ./`

`npm install`

`cd web`

`npm install`

开启本地webpack dev server,运行端口8080
`npm run serve`

本地模拟生产环境打包：
`npm run dev-build`

生产环境打包：
`npm run build`

`cd ..`

开启本地测试接口koa服务,运行端口3000
`npm run dev`

开启服务器生产服务器
`npm run prd`
